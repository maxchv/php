При выполнении заданий использовать функции регулярных выражений

1. Выделите жирным все переменные PHP, встречающиеся в тексте.

Регулярка должна пройти следующие тесты:

Дана строка: "текст $var текст".
Результат должен быть таким: "текст <b>$var</b> текст".

Дана строка: "текст $var1 текст".
Результат должен быть таким: "текст <b>$var1</b> текст".

Дана строка: "текст $var_var текст".
Результат должен быть таким: "текст <b>$var_var</b> текст".

Дана строка: "текст $var1["key1"] + $var2["key2"] текст".
Результат должен быть таким: "текст <b>$var1["key1"]</b> + <b>$var2["key2"]</b> текст".

2. Удалить все слова из предложения, содержащие две одинаковые следующие друг за другом буквы, причем так, 
чтобы не осталось пробелов. Считайте все слова состоящими из маленьких латинских букв.


Регулярка должна пройти следующие тесты:

Дана строка: "wword word".
Результат должен быть таким: "word".

Дана строка: "word wordd".
Результат должен быть таким: "word".

Дана строка: "word worrd".
Результат должен быть таким: "word".

Дана строка: "word worrd word".
Результат должен быть таким: "word word".

Дана строка: "word word woord".
Результат должен быть таким: "word word".

Дана строка: "wword word word".
Результат должен быть таким: "word word".

3. Переставить местами буквы до @ и после нее.

Дана строка: "aa@bb".
Результат должен быть таким: "bb@aa".
Дана строка: "aaA@Bbb".
Результат должен быть таким: "Bbb@aaA".
Дана строка: "a@bbbbbbbb".
Результат должен быть таким: "bbbbbbbb@a".
Дана строка: "2a@bbbbbbbb".
Результат должен быть таким: "2bbbbbbbb@a".
Дана строка: "a@bbbbbbbb2".
Результат должен быть таким: "bbbbbbbb@a2".
Дана строка: "a2@bbbbbbbb".
Результат должен быть таким: "a2@bbbbbbbb"
