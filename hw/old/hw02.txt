Домашнее задание.

В справочнике должны присутствовать функции представленные на странице http://php.net/manual/ru/ref.array.php.

Необходимо реализовать поиск функции по названию.
К input добавить datalist со списком всех функций для автодополнения.

Для найденной функции выводить следующую информацию:
	1. Описание
	2. Список параметров 
	3. Возвращаемые значения
	4. Ошибки 
Указанные данные о функциях хранить в многомерном массиве
